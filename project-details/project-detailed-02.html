Lottery Game: Unity 3D Project

This lottery game is developed in Unity 3D, featuring an interactive betting system where players select numbers from 00 to 99 and place bets using in-game currency. The game operates with three daily draws, ensuring fair results through a random number generator (RNG) algorithm. Players can track past results, view transaction history, and participate in upcoming draws through an intuitive UI/UX design powered by TextMeshPro.

To ensure security and authentication, the game integrates Firebase for user account management, real-time data synchronization, and secure transactions. Users can register, log in, or play as guests, with full access granted upon signup. The admin panel provides game operators with control over bet placements, result management, and player monitoring. A built-in wallet system allows users to buy, earn, and withdraw in-game currency, enhancing player engagement.

The game features rewarded ads, in-app purchases, and jackpot events to sustain monetization while keeping gameplay exciting. Animated draw sequences, celebratory effects, and notifications enhance player immersion. The Firebase database ensures real-time updates for bets, lottery results, and transaction records, offering a seamless experience across devices.

This project showcases expertise in Unity game development, Firebase integration, UI/UX design, and mobile optimization. Future updates may introduce multiplayer competitions, leaderboard rankings, and special event lotteries to keep the game fresh and engaging.
